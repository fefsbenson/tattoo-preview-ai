Blackwork
Tattoo AIGuia de Execução

Um web app que gera tatuagens blackwork com IA — e deixa o cliente se ver tatuado antes de fechar o serviço. Clone, rode e continue a execução com os agentes do AIOX.

00

O Que É

App single-page em Next.js 14 + TypeScript + Tailwind que gera arte de tatuagem blackwork (100% tinta preta, sem cor) via OpenAI gpt-image-1. Roda em localhost:3000. Dois modos:

Gerar do Zero

Descreva a tatuagem em texto e receba um design flash-sheet pronto em preto sólido.

Ver-se Tatuado

Envie uma foto sua — a IA compõe a tatuagem na sua pele, seguindo o contorno do corpo.

01

Rodar o App

Precisa de Node.js 18+, npm e uma chave da OpenAI com créditos e acesso ao gpt-image-1.

terminal — setup
# 1. Clone o repositório
$ git clone https://github.com/fefsbenson/tattoo-preview-ai.git
$ cd tattoo-preview-ai

# 2. Instale as dependências
$ npm install

# 3. Configure sua chave da OpenAI
$ cp .env.local.example .env.local
#    edite .env.local → OPENAI_API_KEY=sua-chave-aqui

# 4. Suba o servidor
$ npm run dev

Abra localhost:3000. Obtenha a chave em platform.openai.com/api-keys.

Reinicie após editar a chave. O Next.js só carrega o .env.local na inicialização — rode npm run dev de novo após preencher a chave.
02

Continuar no AIOX

Este MVP nasceu de uma orquestração de agentes do AIOX open source. Com o AIOX instalado, você clona o projeto pro seu workspace e continua evoluindo com os mesmos agentes que o criaram — sem começar do zero.

terminal — workspace aiox
# Instale o AIOX open source (se ainda não tiver)
$ npx aiox-init

# Traga o MVP pro seu workspace
$ git clone https://github.com/fefsbenson/tattoo-preview-ai.git
$ cd tattoo-preview-ai && npm install
$ cp .env.local.example .env.local   # preencha OPENAI_API_KEY

03

Regras da Casa

  • Nunca commite .env.local. A chave só é lida server-side em api/generate/route.ts — jamais chega ao navegador.
  • Custo real: cada geração consome créditos da OpenAI (~US$ 0,04–0,07/imagem em medium). Ajuste quality em lib/openai.ts.
  • Sem login, sem persistência — as imagens não são armazenadas no servidor.
  • A tatuagem em foto é uma prévia aproximada — não é o resultado final na pele.
